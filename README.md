# ⚛️ React Fundamentals

This repository is for learning and practicing the core concepts of React. It includes fundamental notes, code snippets, and basic beginner-friendly applications.

## 🚀 Core Concepts Covered

- **JSX:** Understanding how to write HTML structures directly inside JavaScript.
- **Components:** Building reusable functional components to create modular User Interfaces.
- **Props:** Passing dynamic data from parent components down to child components.
- **State Management (useState):** Handling local component state and managing dynamic UI updates.
- **Side Effects (useEffect):** Fetching data, setting up listeners, and managing component lifecycles.
- **Conditional Rendering:** Displaying UI elements selectively using ternary operators and logical structures.
- **List Rendering:** Rendering arrays efficiently using the `.map()` method with unique keys.

### How to Run the Project
To spin up the development servers, open your terminal, navigate to the specific project directory, and execute the corresponding command.

* **Vite + React Setup:**
  ```bash
  cd Project_Folder
  npm create vite@latest 
  npm run dev
  ```
* **React Setup:**
  ```bash
  cd Project_Folder
  npm create-react-app
